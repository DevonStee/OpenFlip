# Skills Directory

This directory contains skills that extend the agent's capabilities.

## Structure

- **`public/`**: System preset public skills.
- **`private/`**: Private or internal tool configurations.
- **`user/`**: User-created custom skills.

## Managing Skills

To create a new skill, please refer to the **Skill Creator** skill located in `user/skill-creator`.
It provides guidelines on how to structure and place new skills.

## Current Setup

All existing user skills have been moved to the `user/` directory.
